# APR Hub Dashboard Design

A modern and responsive dashboard UI built using **React**, **TypeScript**, **Vite**, and **Radix UI** components.  
This project is based on the APR Hub Dashboard Figma design.

---

## Tech Stack

- **React 18**
- **TypeScript**
- **Vite**
- **Radix UI**
- **Tailwind CSS**
- **Lucide Icons**
- **Recharts**
- **React Hook Form**
